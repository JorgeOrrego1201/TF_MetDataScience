## Tema elegido y problema a resolver

Se tomará el caso hipotético de una startup que quiere irrumpir en el mercado del streaming de música, pero necesita de un servicio que pueda igualar en algunos aspectos del negocio a empresas grandes como Spotify, Apple y Google si es que quiere mantenerse en el tiempo. Para eso, la empresa necesita desarrollar un sistema o modelo de recomendación de música que pueda inferir a partir de los gustos musicales de los clientes qué canciones o álbumes les podría llamar la atención. 

El objetivo principal del proyecto es el incrementar el número de suscriptores de la plataforma de streaming de música de la startup, lo suficiente para seguir en el mercado y para competir con las demás empresas a largo plazo. Para esto, se desarrollará un modelo de recomendación musical con machine learning, el cual se integrará como una funcionalidad extra para retener y atraer a nuevos clientes.

## Roles 

Como el trabajo es de solo un integrante, este tendrá que ejercer los roles de Data Engineer para la preparación de los datos junto al de Data Scientist para el análisis avanzado de los datos y la creación del modelo de Machine Learning.
